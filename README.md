# Global Flutter KMP Project

Этот проект содержит агентов и субагентов для работы с Claude и другими системами.

## Описание

Проект реализует архитектуру агентов и субагентов, предназначенных для взаимодействия с различными AI-системами, включая Claude. Архитектура позволяет создавать сложные цепочки обработки задач с использованием специализированных агентов и их иерархий.

## Компоненты

- **Агенты**: Основные исполнители задач
- **Субагенты**: Специализированные компоненты, работающие под управлением основных агентов
- **Интеграции**: Подключение к различным AI-платформам (Claude и др.)

## Использование

Проект предназначен для разработки сложных AI-систем с иерархической архитектурой агентов.
